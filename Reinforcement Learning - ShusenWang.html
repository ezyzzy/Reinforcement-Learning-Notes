Reward & Return

  • Return U & reward R

回报是未来的奖励总和，越大越好。

未来的奖励没有现在的奖励值钱，所以需要打个折扣。Discounted

hyperparameter discounted rate γ

为什么U和R用大写字母，因为还未被观察，是随机事件。

意思就说每一步policy取决于此刻的Return。

 

Value Function

U 是时刻t后的奖励总和的随便变量

Qπ 是时刻t后的奖励总和的期望

Q∗ 是时刻t后找到最优策略π后的最大化奖励总和的期望

Vπ 是时刻t时不同action对应的Qπ的期望

  • Action-value function Qπ

Ut 是个随机变量，Qπ 是期望。

  • Optimal action-value function Q∗

时刻 t 找到最好的 policy π 让 Q 最大化

  • State-value function 状态价值函数 Vπ

用于判断当前局势好不好

  • action & state - value function

前者评价 action，后者评价 situation， 后者的期望评价 policy
